## Overview 

Adding and renewing [wildcard SSL certificates](/:product/:version?/security/ssl#wildcard-certificates) relies on Cloud 66 having API access to your DNS provider. To set up a DNS provider, follow the instructions below. 

If your provider is not listed, consider using [RFC 2136](https://www.rfc-editor.org/rfc/rfc2136) if your provider supports the protocol.

## Instructions per DNS provider

Each DNS provider has specific credentials that need to be created and then [added to your Cloud 66 account](/:product/:version?/security/ssl#step-1-set-up-a-dns-provider). Find your provider below for more details.
